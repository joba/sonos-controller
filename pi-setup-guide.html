Sonos App — Raspberry Pi 5 Setup Guide

This guide walks you through hosting your Next.js Sonos controller app on a Raspberry Pi 5 running Ubuntu Server 24.04 LTS.

Kit here

What you need

  • Raspberry Pi 5 (8GB)
  • Kingston 64GB microSD card
  • Raspberry Pi 5 power supply (USB-C)
  • Another computer to flash the SD card
  • Your project files (or Git access to your repo)

Step 1 — Flash Ubuntu Server to the SD card

  1. Download and install Raspberry Pi Imager from raspberrypi.com/software
  2. Insert your SD card into your computer
  3. Open Raspberry Pi Imager and choose:
    • Device: Raspberry Pi 5
    • OS: Other general-purpose OS → Ubuntu → Ubuntu Server 24.04 LTS (64-bit)
    • Storage: your SD card
  4. Click Next, then Edit Settings to configure:
    • Set a hostname (e.g. sonos-pi)
    • Set a username and password
    • Enable SSH (under the Services tab)
    • Optionally configure Wi-Fi (Ethernet recommended)
  5. Click Save, then Yes to write the image

Step 2 — Boot and connect

  1. Insert the SD card into the Pi and connect it to your network via Ethernet
  2. Plug in the power supply — it will boot automatically
  3. Wait ~60 seconds for first boot to complete
  4. Find the Pi's IP address on your router, or try: 
    ping sonos-pi.local
  5. SSH into the Pi: 
    ssh your-username@sonos-pi.local

Step 3 — Update the system

sudo apt update && sudo apt upgrade -y

Step 4 — Install Node.js via nvm

Do not use apt install nodejs — the version is outdated. Use nvm instead. 
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts
node -v   # should print v22.x or later

Step 5 — Copy your project to the Pi

If your project is in a Git repository:

git clone https://github.com/your-username/your-repo.git ~/sonos
cd ~/sonos

Or copy files from your Mac using scp:

# Run this on your Mac, not the Pi
scp -r /Users/jonbagling/Documents/jobb/sonos your-username@sonos-pi.local:~/sonos

Step 6 — Set up environment variables

cd ~/sonos
cp .env.example .env.local   # if an example file exists
nano .env.local               # add your Spotify API keys etc.

Step 7 — Install dependencies and build

cd ~/sonos
npm install
npm run build
The first build may take a few minutes on the Pi. Subsequent builds are faster.

Step 8 — Install pm2 and start the app

pm2 keeps your app running in the background and restarts it after reboots. 

npm install -g pm2
pm2 start npm --name sonos -- start
pm2 save
pm2 startup

The pm2 startup command will print a sudo command — copy and run it to enable autostart on boot.

Step 9 — Verify it's running

pm2 status           # should show "sonos" as online
pm2 logs sonos       # view live logs

Open a browser on the same network and go to: http://sonos-pi.local:3000

Step 10 — Optional: use port 80

To access the app at http://sonos-pi.local without typing :3000, use a port redirect: 

sudo iptables -t nat -A PREROUTING -p tcp --dport 80 -j REDIRECT --to-port 3000
sudo apt install iptables-persistent -y
sudo netfilter-persistent save

Useful commands

pm2 restart sonos    # restart the app
pm2 stop sonos       # stop the app
pm2 logs sonos       # view logs
pm2 monit            # live process monitor

Updating the app

cd ~/sonos
git pull             # if using Git
npm install
npm run build
pm2 restart sonos
